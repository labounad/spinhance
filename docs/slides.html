Spinhance
A 90 MHz benchtop spectrum looks blurry — but the blur isn't noise. It's the full spin system, just scrambled.

We train a network to invert that blur back into field-independent parameters (δ, J, degeneracies), so a cheap spectrum can be re-simulated exactly at any field.

The problem & the bet

High-field structure, locked behind a $1M magnet.

At low field the chemical-shift spread Δν shrinks toward J, so multiplets overlap — the spectrum goes second-order. Peak-picking fails, and benchtops get dismissed as "low information." They aren't.

How it works · one molecule, four modules

SMILES → spin-system matrix → spectra → back to the matrix.

⚗️1 · GENERATESam

Label every proton

SMILES → 3D, every proton tagged by spin group — distinguishing hard equivalence (same shift, one group) from soft (same shift, separate groups). RDKit embedding + a deuterium-substitution magnetic-equivalence test.

69,324 exact-8-group ChEMBL molecules; one scan buckets all counts (1–10) at once.

🧮2 · SPIN SYSTEMYiming

Shifts δ & couplings J

Molecule → parameters: δ via NMRShiftDB2, J via Karplus + literature tables, a dedicated path per mechanism (geminal, vicinal, benzylic, allylic, aromatic…). Predictor spread stored once, so spectra randomize without re-running it.

Nearly free, high quality: a 2M+ molecule dataset generated in a weekend.

📡3 · SIMULATELucas

Exact spin Hamiltonian

Diagonalize the spin Hamiltonian → realistic spectra at 90 & 600 MHz, capturing the higher-order coupling that makes low-field work. Started by scripting MestReNova; then rebuilt from scratch as pyspin.

pyspin reproduces MestReNova to r ≈ 0.999 — open-source, license-free, faster.

🧠4 · MODELSam

Spectrum → matrix

One 90 MHz spectrum → field-independent matrix. ResNet-1D → 4 typed heads (δ, |J|, J-presence, degeneracy). A canonical sort resolves the S₈ label-permutation problem; GroupNorm + EMA stabilize training.

Trained on AWS (L40S) over 2M+ PubChem; leakage-safe scaffold + matrix-dedup splits.

Impact · the tool · the team

Democratizing structure verification.

  • Why it matters: structure verification is the daily bottleneck of synthetic & medicinal chemistry — gating it behind million-dollar magnets gates who gets to do rigorous chemistry.
  • Biomedical tie: faster, cheaper structure confirmation accelerates small-molecule discovery — the front end of drug discovery.
  • A real, open tool: public repo, README, validated license-free simulator — clone and run.
  • How it scales — autoai: a two-level, human-out-of-the-loop orchestrator — an Opus "scientist" picks strategies, a Sonnet "worker" writes & trains, a Sonnet "auditor" checks the claims. Nascent, not the headline — but it's how this improves without us.
